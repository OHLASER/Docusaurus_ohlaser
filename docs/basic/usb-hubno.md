# USB-HUBの選定

PCのスペックによりますが、特にノートPCやコンパクトPCの場合に、USBケーブルに対する電力供給が足りないことがあります。

PCのUSB通信が不安定な場合はセルフパワーUSB-HUBの増設をお勧めします。
