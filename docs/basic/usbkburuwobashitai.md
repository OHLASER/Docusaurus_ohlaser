# USBケーブルを伸ばしたい場合

HAJIMEには新規ご購入の際に2ｍのUSBケーブルが付属しています。

原則、通信障害が起こりやすくなる為推奨はしておりませんが、どうしても延長したい場合は純正ケーブルと延長ケーブル（2ｍまで）の間に電力供給型のHUBを挟んで使用していただきますようお願いいたします。
