モーションサービスの再起動

モーションサービスの再起動方法​

HARUKAを閉めて頂いている状態で Windowsプログラムグループ内の

HARUKAフォルダの「サービス管理」を起動します。

起動後に左上にある「解除」ボタンを押します。（途中でユーザー制御の画面が表示しましたら、「実行」をクリックしてください。）

「登録」ボタンを押します。（途中でユーザー制御の画面が表示しましたら、「実行」をクリックしてください。）
