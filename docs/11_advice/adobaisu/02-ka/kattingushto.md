# カッティングシート

![](/assets/smls1.jpg)

一般的なカッティングシートは塩ビ系素材なので、レーザーでは加工できません。カッティングプロッターをご利用ください。

オレフィン系素材のカッティングシートであればレーザーでの加工が可能です。
