# 加工機:未接続と接続が繰り返し表示される

## 症状

* HARUKAの電源マークアイコンがついたり消えたりする
* 接続中と未接続を繰返し表示する
* USB 接続の音が繰返し鳴る



## 解決策

上記の症状が生じる場合、下記解決方法を参考し、試してください。
