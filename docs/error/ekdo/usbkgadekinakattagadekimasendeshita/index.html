暗号化USBキーが検出できなかった為、初期化ができませんでした。

症状​

考えられる原因​

1. キードングルがささっているか確認してください。​

ドングルがPCにささっている場合、差し込んでいるポートにグラつきなどが無いか、別のポートに差し込んでいただきドングルを認識をするか確認をお願い致します。

2. USB暗号化キー（ドングル）ドライバーがPCに入っていない。​

下記ページの手順をご参照ください。
