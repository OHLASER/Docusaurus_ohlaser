# 転送できない



:::note
[データが開けない](/docs/soft/dtawomimenai)
:::

## 直接読み込みできるが、転送ができない。

プラグイン動作対応バージョンの違いによるものです。

:::caution
* Illustrator プラグイン・・・・CS6～CC2019まで
* CorelDRAW プラグイン・・・X5～X8まで（Essentials 除く）
:::

:::success
対応バージョンに変えるか、HARUKA から直接開いてください。
:::
