# ゴム板を加工した場合

ゴム板の粉塵や汚れはアルコールや濡れた布巾、ウエットティッシュでは落ちません。

キッチン用のアルカリ性の洗剤（マジックリン）と駆動部品を掃除する際に産業用ワイパー（キムワイプ）を御用意ください。

※洗剤で掃除した後は必ずアルコールで拭いてください
